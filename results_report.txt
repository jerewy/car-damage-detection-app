## Why This Will Work Better

| Old Approach | New Approach | Benefit |
|--------------|--------------|---------|
| Custom DIVs with inline styles | Bootstrap `.progress` class | Uses Bootstrap's built-in styles |
| Fixed 100px width | `flex: 1` (responsive) | Adapts to cell width |
| Plain text percentage | Badge with `.badge` class | Better visual styling |
| Complex color logic | Bootstrap classes (`bg-success`, `bg-warning`) | More reliable |
